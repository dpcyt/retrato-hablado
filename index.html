Pantalla inicial de Stable Diffusion: Al iniciar el programa de Stable Diffusion, nos encontramos con una interfaz que permite generar imágenes a partir de distintos parámetros. Para crear una imagen de acuerdo a nuestras preferencias, se deben seguir los siguientes pasos:

  1. Dirigirse a img2img para cargar la imagen
  2. Crear un prompt (en inglés)
  3. Hacer uso de Lora
  4. Configuración de tamaño de imagen, CFG Scale y Denoising Strength
  5. Configurar Tiled Diffusion
  6. Configurar ControlNet

1 - Dirigirse a img2img para cargar la imagen
Nos dirigimos a la opción "img2img". Esta función nos permite trabajar con una imagen de referencia, utilizando la inteligencia artificial para modificarla o mejorarla según nuestras preferencias. A continuación, cargamos la imagen que deseamos modificar o usar como base para generar nuevas imágenes

2 - Crear un prompt (en inglés)
El prompt es una descripción detallada de lo que deseas generar o los aspectos que más quieras destacar en la imagen. Este texto guía al modelo para producir un resultado acorde a tus expectativas. Asegúrate de que esté en inglés. Si no dominas el inglés, puedes utilizar traductores en línea como apoyo para redactar tu descripción.

3 - Hacer uso de Lora
Lora es una herramienta que permite agregar detalles adicionales o ajustar características específicas en la imagen generada. Es especialmente útil para lograr un mayor nivel de personalización y precisión en el resultado final.

Los modelos Lora utilizados son los siguientes:

  1. SDXLrender: Este modelo está diseñado para crear imágenes de alta calidad con un enfoque realista. Es ideal para generar personajes de fantasía, paisajes y escenas modernas con una estética fotorrealista. Ofrece un sistema mejorado de detalles de fondo y colores más vivos. Se recomienda un peso de 0.2 a 0.9 para lograr un equilibrio entre los detalles deseados y el estilo del modelo.
  2. Add More Details: Este modelo se enfoca en estilos de cómic y arte fantástico, creando ilustraciones semirrealistas, conceptos de personajes y paisajes complejos en 2D o 2.5D. Es ideal para producciones con estética de cómics occidentales y combina bien con otros LoRAs. Se recomienda un peso de 0.5 a 1 para obtener resultados óptimos en estilos de cómic y fantasía.

4 - Configuración de tamaño de imagen, CFG Scale y Denoising Strength
En las configuraciones iniciales, vamos a configurar de la siguiente manera, teniendo en cuenta lo que refiere cada opción:

  • Sampling method: Es el algoritmo que utiliza Stable Diffusion para generar las imágenes a partir del ruido inicial. Elegimos la opción "DPM++ 3M SDE" porque es eficiente y produce resultados de alta calidad, balanceando bien entre velocidad y precisión en la generación de imágenes.
  • Sampling steps: Determinan cuántas iteraciones realizará el modelo para generar la imagen. A más pasos, mayor será el nivel de detalle y calidad, pero también aumentará el tiempo de generación. En este caso, le damos un valor de 18, que es adecuado para obtener buenos resultados sin alargar demasiado el proceso.
  • Tamaño de la imagen: Configuramos el tamaño de la imagen de salida con las opciones Width (ancho) y Height (alto). Estas opciones se definen en píxeles y es importante elegir un tamaño adecuado según el tipo de imagen que queramos generar, ya que esto afecta tanto la calidad como el tiempo de procesamiento.
  • CFG Scale: Controla qué tan estrictamente el modelo sigue el prompt (la descripción que damos). Un valor más alto significa que el modelo será más preciso en seguir las indicaciones del prompt, pero a veces puede perder algo de creatividad. Lo dejamos en 6, que es un valor equilibrado, aunque podemos variar entre 6 a 9 dependiendo de qué tan preciso o creativo queramos que sea el resultado.
  • Denoising strength: Controla cuánto se puede alterar la imagen original, basándose en el nivel de ruido. Con un valor más alto, la imagen puede cambiar más respecto a la original. Lo dejamos en 0.35, un valor moderado. Podemos variar entre 0.3 a 0.9 para obtener diferentes efectos de modificación, siendo más bajo cuando queremos que la imagen final se parezca más a la original.

5 - Configura Tiled Diffusion
Esta opción es útil para generar imágenes de gran tamaño o patrones repetitivos. Ajustando esta opción, puedes crear imágenes más grandes sin perder calidad o coherencia.

Los pasos para configurarlo son los siguientes:

  1. Tilda la opción "Enable Tiled Diffusion": Esta opción activa la función de difusión en mosaico, permitiendo generar imágenes más grandes dividiéndolas en secciones.
  2. Tilda la opción "Keep input image size": Al habilitar esta opción, se asegura que el tamaño de la imagen original se mantenga, sin ser modificado durante el proceso de difusión.
  3. Latent tile width: Define el ancho de los "bloques" latentes en los que se divide la imagen. Configura el valor a 112.
  4. Latent tile height: Establece la altura de los bloques latentes en los que se divide la imagen. Configura el valor a 144.
  5. Latent tile overlap: Controla cuánto se superponen los bloques latentes para mantener la coherencia de la imagen entre las secciones. Configura el valor a 4.
  6. Latent tile batch size: Define cuántos bloques latentes se procesan en cada ciclo, lo que afecta la rapidez y la calidad del procesamiento. Configura el valor a 8.

6 - Configura ControlNet
ControlNet te proporciona un control avanzado en la generación de imágenes, permitiéndote dirigir el modelo de manera más precisa en aspectos como la estructura y la pose de la imagen.

Los pasos para configurarlo son los siguientes:

  1. Habilitar "Enable" y "Pixel Perfect": Activa estas opciones para usar ControlNet y asegurar que la precisión de los píxeles sea la máxima posible.
  2. Tilda en las opciones Control Type "Tile": Elige "Tile" para aplicar el control en secciones o mosaicos.
  3. Verifica que en Preprocessor esté "tile_resample"
  4. Verifica que en Model esté "control_v11f1e_sd15_tile"
  5. En Control Weight: La recomendación es variar este parámetro entre 0.3 a 1.6. Para obtener un buen equilibrio entre control y calidad, configúralo en 0.6.

Generación de Imágenes con Configuraciones Personalizadas

Una vez que tengas todas las configuraciones listas, es momento de comenzar a generar imágenes con los parámetros que desees. La imagen generada actuará como un feedback, permitiéndote verificar si el resultado se ajusta a lo que imaginas o deseas. Para comenzar, los parámetros con los que debemos experimentar son:

  1. El Prompt: Cuanto más detallado sea el prompt, mejor podrá interpretar el programa lo que deseas.
  2. CFG Scale: Este parámetro controla estrictamente el comportamiento del prompt, ajustando la influencia de las instrucciones dadas.
  3. Denoising strength: Controla cuánto se puede modificar la imagen original, permitiendo más o menos alteraciones durante la generación.

Una vez que experimentes con estos valores, podrás continuar ajustando otros parámetros como:

  • Control Weight: Este parámetro influye en el control que se ejerce sobre la imagen generada.
  • Modelos Lora: Puedes modificar el peso de los modelos Lora para agregar detalles o personalización adicional.
